## Questions Solved

### 1. Dijkstra’s Algorithm

When actions have different costs in a state-based search space, use best-first search where the evaluation function is the path cost from the root to the current node. This is known as Dijkstra’s algorithm (Uniform Cost Search). Implement this algorithm for cities in India using road distances.

---

### 2. UGV Navigation (Static Environment)

Design an algorithm for a UGV to navigate a 70×70 grid with known obstacles. The obstacle density is randomly generated at three levels. The UGV must find the shortest path from start to goal and compute Measures of Effectiveness.

---

### 3. UGV Navigation (Dynamic Environment)

Extend the above problem by considering dynamic obstacles that are not known beforehand. Design an approach for the UGV to navigate and find the optimal path in such conditions.
